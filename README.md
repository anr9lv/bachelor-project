#  ProDash: Agile Project Management Dashboard

A modern, full-stack web application designed to help IT teams and developers plan, track, and visualize their project workflows using the Kanban methodology. Built with the MERN stack and modern UI/UX principles.

---

##  Overview

Managing project tasks efficiently is crucial for team success. ProDash provides a centralized workspace where users can generate tasks, assign deadlines, and move them through a structured lifecycle (To Do -> In Progress -> Done). The application features a dynamic dashboard that calculates project health in real-time and visualizes data using interactive charts.

---

##  Features

*   **Interactive Kanban Board:** Easily update task statuses and organize workflow.
*   **Data Visualization:** Real-time progress bars, pie charts, and bar charts powered by Recharts.
*   **Modern SPA Routing:** Seamless page transitions between the Dashboard and Settings using React Router.
*   **Premium UI/UX:** Built with Tailwind CSS v4 featuring glassmorphism elements, soft gradients, and responsive layouts.
*   **Secure Backend API:** RESTful API architecture equipped with bcrypt password hashing and JWT token generation.

---

##  Tech Stack

### Frontend (Client)
*   **React.js** (via Vite for lightning-fast HMR)
*   **Tailwind CSS v4** (Styling & Layout)
*   **React Router DOM** (Multi-page navigation)
*   **Recharts** (SVG-based charting library)
*   **Axios** (HTTP client for API requests)

### Backend (Server)
*   **Node.js & Express.js** (Server environment & API routing)
*   **MongoDB & Mongoose** (NoSQL database & Object Data Modeling)
*   **Bcrypt.js** (Cryptographic password hashing)
*   **JSON Web Tokens (JWT)** (Stateless authentication)
